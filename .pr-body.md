## Summary

- Replaces the static SVG pixel logotype in the site footer with an interactive text-based `SiteFooterInteractiveLogotype` component
- Adds hover-driven gradient reveal animation using a radial mask that follows the cursor, with a crosshair cursor on hover
- Iterates on layout and sizing (viewBox, text positioning, 3rem type scale) and simplifies mouse tracking by calculating mask position directly in the event handler
- Updates `SiteFooter` to render the new logotype (YASH 14) in a centered container and removes the extra bottom spacer
